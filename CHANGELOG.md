# 更新日志 Changelog

所有本项目的重要变化都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-11-11

### ✨ Added（新增）

#### 核心功能
- 🎨 完整的HTML5 Canvas绘画功能
  - 支持鼠标和触摸操作
  - 400x400像素画布
  - 流畅的绘画体验

- 🎯 可调节的绘画工具
  - 画笔大小范围：1-20像素
  - 8种预设颜色（黑、红、绿、蓝、黄、紫、青、橙）
  - 实时画笔预览

- 🤖 AI猜测系统
  - 基于图像特征的智能识别
  - 显示前3个最可能的猜测
  - 置信度百分比显示
  - 异步处理和加载动画

- 📊 历史记录和统计
  - 猜测历史记录（最多保存10条）
  - 总猜测次数统计
  - 当前工具状态显示
  - 时间戳记录

- 🎮 用户界面
  - 左右分栏布局
  - 响应式设计
  - 现代化的UI风格
  - 流畅的动画和过渡效果

#### 技术特性
- 纯前端实现，无需后端
- 模块化的JavaScript代码
- 完整的事件处理
- 跨浏览器兼容性

#### 文档
- 详细的README说明
- 使用指南和技巧
- 贡献指南
- MIT开源许可证

#### 开发工具
- GitHub工作流自动部署
- .gitignore配置
- 本地启动脚本

### 🎯 识别对象库

支持识别以下物体：
- 基础形状：圆形、方形、三角形、星形、心形
- 自然物体：太阳、树、花、鸟、鱼
- 人造物品：房子、车、书、杯子、手机
- 生物：人、猫、狗、笑脸
- 食物：苹果等水果

### 🌐 浏览器支持

- ✅ Chrome (最新版)
- ✅ Firefox (最新版)
- ✅ Safari (最新版)
- ✅ Edge (最新版)
- ✅ 移动浏览器 (iOS Safari, Chrome Mobile)

## 如何升级

从旧版本升级到新版本：

```bash
# 拉取最新代码
git pull origin main

# 刷新浏览器（清除缓存）
# 按 Ctrl+Shift+R (Windows/Linux) 或 Cmd+Shift+R (Mac)
```
