# 🎨 你画我猜 Draw & Guess 游戏

一个有趣的网页绘画猜测游戏，用户在左侧画板上绘画，AI会尝试猜测用户画的内容。

## 🎮 游戏特色

### 主要功能
- **左侧画板区域**：提供完整的绘画工具
  - 400x400像素的绘画画布
  - 可调节画笔大小（1-20px）
  - 8种预设颜色选择
  - 清空画布功能
  - 支持鼠标和触摸操作

- **右侧AI猜测区域**：智能内容识别
  - 基于图像特征的AI猜测算法
  - 显示多个可能的猜测结果和置信度
  - 猜测历史记录
  - 统计信息显示

### 技术特点
- 纯前端实现，无需后端服务器
- 响应式设计，支持移动设备
- 流畅的绘画体验
- 智能的形状和特征识别算法

## 🚀 使用方法

### 启动游戏
1. 直接用浏览器打开 `index.html` 文件
2. 或者使用本地服务器：
   ```bash
   # 使用Python简单服务器
   cd ~/front_end/game/bpit
   python3 -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

### 游戏操作
1. **开始绘画**：在左侧画板上点击并拖拽鼠标进行绘画
2. **调整工具**：
   - 使用滑块调节画笔大小
   - 点击色块选择画笔颜色
3. **获取猜测**：画完后点击"开始猜测"按钮
4. **查看结果**：右侧会显示AI的猜测结果和置信度
5. **清空重画**：点击"清空"按钮开始新的绘画

## 🤖 AI识别算法

游戏内置了智能的图像识别算法，能够识别以下类型的绘画：

### 识别对象库
- **基础形状**：圆形、方形、三角形、星形、心形
- **自然物体**：太阳、树、花、鸟、鱼
- **人造物品**：房子、车、书、杯子、手机
- **生物**：人、猫、狗、笑脸
- **食物**：苹果等水果

### 识别特征
- **形状分析**：检测圆形和角形特征
- **覆盖率分析**：分析绘画的复杂程度
- **颜色多样性**：考虑使用的颜色种类
- **置信度计算**：为每个猜测提供可信度评分

## 🎨 绘画技巧

为了获得更好的识别效果，建议：

1. **画得清晰**：使用较粗的画笔，确保线条清楚
2. **突出特征**：强调物体的主要特征
3. **简单明了**：避免过于复杂的细节
4. **完整图形**：尽量画出完整的物体轮廓

## 🌟 推荐绘画主题

### 简单物体
